 📊 PulseIQ - Sales Dashboard

📌 Project Overview

This project is an interactive sales dashboard built using Streamlit.
It helps in analyzing sales data and provides insights through visualizations and key metrics.

---

🚀 Features

- 🔍 Data filtering (Region, Category)
- 📊 Sales and Profit visualization
- 📈 KPI metrics (Total Sales, Profit, Orders)
- ⬇ Download filtered data
- 🤖 Basic prediction module (demo)

---

🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit

---

📂 Dataset

- Superstore Sales Dataset
- Contains customer orders, sales, profit, and region details

---

▶️ How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run the app:

streamlit run app.py
